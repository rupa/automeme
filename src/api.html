Official Auto-Meme Enterprise API

All you have to do is GET one of the following URLs:

  • http://meme.boxofjunk.ws/moar.txt
  • http://meme.boxofjunk.ws/moar.html
  • http://meme.boxofjunk.ws/moar.txt.json
  • http://meme.boxofjunk.ws/moar.html.json

Differences between .txt and .html:

.txt .html
-- &#8212; (—)
_FOO_ <em>FOO<em>

Ten memes are returned by default. To change this, append ?lines=n to the URL, where n is in the range 1–20.

Don't hammer the server plz thx.

Diving in

From a shell

$ curl -s meme.boxofjunk.ws/moar.txt?lines=1

Python

>>> import urllib
>>> url = 'http://meme.boxofjunk.ws/moar.txt'
>>> memes = urllib.urlopen(url).read().rstrip().split("\n")
>>> print memes.pop()

Ruby

>> require "open-uri"
>> url = "http://meme.boxofjunk.ws/moar.txt"
>> memes = open(url).read.split("\n")
>> puts memes.pop
