Official Auto-Meme Enterprise API

All you have to do is GET one of the following URLs:

  • http://meme.boxofjunk.ws/moar.txt
  • http://meme.boxofjunk.ws/moar.txt.json
  • http://meme.boxofjunk.ws/moar.html
  • http://meme.boxofjunk.ws/moar.html.json

These return 10 memes by default. To change this, append ?lines=X to the URL, where X is in the range 1–20.

Don't hammer the server plz thx.

Diving in (the lazy way)

#!/bin/sh
curl -s meme.boxofjunk.ws/moar.txt | head -n1
#!/usr/bin/env python
from urllib import urlopen
print urlopen('http://meme.boxofjunk.ws/moar.txt').read().split('\n')[0]
#!/usr/bin/env ruby
require "open-uri"
puts open("http://meme.boxofjunk.ws/moar.txt").read.split("\n").first
