Official Auto-Meme Enterprise API

(Not to be confused with the Yahoo Meme API.)

All you have to do is GET one of the following URLs:

  • http://api.automeme.net/moar.txt
  • http://api.automeme.net/moar.html
  • http://api.automeme.net/moar.txt.json
  • http://api.automeme.net/moar.html.json

Differences between .txt and .html:

.txt .html
-- &mdash; (—)
_FOO_ <em>FOO</em>

Ten memes are returned by default, separated by new line characters. To change the number of memes, append ?lines=n to the URL, where n is a number in the range 1–80. For example: http://api.automeme.net/moar.txt?lines=25

For backwards compatibility, the domain names automeme.net and meme.boxofjunk.ws are also allowed, for now.

Significant changes to the API will be posted to the Auto-Meme blog.

Diving in

From a shell

curl api.automeme.net/moar.txt?lines=1
or
wget -q -O - api.automeme.net/moar.txt?lines=1

Python

import urllib
url = 'http://api.automeme.net/moar.txt'
memes = urllib.urlopen(url).read().rstrip().split('\n')
print memes.pop()

Ruby

require 'open-uri'
url = 'http://api.automeme.net/moar.txt'
memes = open(url).read.split("\n")
puts memes.pop

Perl

use LWP::Simple;
getprint 'http://api.automeme.net/moar.txt?lines=1';
